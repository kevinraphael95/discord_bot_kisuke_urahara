!commandslist

!kisukevol
🌀 Kisuke vole 10% du Reiatsu d'un membre comme un joueur normal.
!reiatsuadmin ADMIN
Gère le Reiatsu : set, unset, change, spawn, speed.
!say_as ADMIN

!say_as <mention|id|pseudo> <message>
!sync
Synchronise les commandes slash (serveur ou global).
!testgame
Tester un mini-jeu via son numéro ou 'all'.
!testtache
