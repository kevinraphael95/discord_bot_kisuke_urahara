Kisuke Bot

Bot Discord Bleach — mini-jeux, Reiatsu, commandes slash.

Présentation

Kisuke Urahara Bot est un bot Discord centré sur l'univers Bleach. Il propose un système de Reiatsu persistant, des mini-jeux, des commandes Bleach et une FAQ interactive du Klub Outside.

Fonctionnalités

💠
Reiatsu
Système de collecte persistant avec classes, skills, vol et quêtes.
⚔️
Bleach
Fiches de personnages, combats, Kidō, scans, et plus.
🕹️
Mini-jeux
Pendu, Mastermind, Pressing, drapeaux, capitales, RPG…
📖
Klub Outside
600+ questions-réponses de la FAQ officielle, navigables ici.
🎲
Fun
Ship, pizza aléatoire, couleur, tri visuel, loterie…
⚙️
Admin
Gestion du Reiatsu, sync des slash commands, panneau admin.

Classes Reiatsu

🥷
Voleur
Cooldown de vol réduit, 67% de réussite, vol garanti en skill.
🌀
Absorbeur
+5 par absorption. Skill : prochain Reiatsu forcément Super (+100).
🎭
Illusionniste
50% de ne pas perdre quand volé. Skill : faux Reiatsu piège.
🎲
Parieur
Gains aléatoires à chaque absorption. Skill : machine à sous.

Installation rapide

1
Cloner le dépôt
git clone https://github.com/kevinraphael95/discord_bot_kisuke_urahara.git
2
Installer les dépendances
pip install -r requirements.txt
3
Configurer .env — DISCORD_TOKEN, DISCORD_APP_ID, COMMAND_PREFIX…
4
Lancer
python bot.py
