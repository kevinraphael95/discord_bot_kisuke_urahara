Guide

Installation, Reiatsu et commandes — tout en un.

Étapes d'installation

1

Prérequis

Assure-toi d'avoir les outils suivants installés sur ta machine.

Python 3.10+ pip Git Compte Discord Dev
2

Cloner le dépôt

Clone le projet depuis GitHub dans ton répertoire de travail.

git clone https://github.com/kevinraphael95/discord_bot_kisuke_urahara.git
cd discord_bot_kisuke_urahara
3

Installer les dépendances

Installe les librairies Python requises via pip.

pip install -r requirements.txt
4

Configurer les variables d'environnement

Crée un fichier .env à la racine du projet et renseigne ces variables.

COMMAND_PREFIXPréfixe du bot
DISCORD_APP_IDID de ton serveur
DISCORD_TOKENToken du bot Discord
GPT_OSS_API_KEYAPI GPT OSS
ADMIN_PASSWORD?
ADMIN_PORT?
ADMIN_SECRET?
5

Lancer le bot

Démarre le bot.

python bot.py

ou

bash start.sh

Comment ça marche

💠

Absorption

Le Reiatsu apparaît régulièrement sur un salon du serveur. Clique sur l'emoji en réaction pour l'absorber avant les autres.

⚡

Super Reiatsu

Un Reiatsu normal rapporte +1. Un Super Reiatsu (rare) rapporte +100.

🩸

Vol de Reiatsu

/reiatsuvol @joueur — 25% de chance de voler 10% du Reiatsu d'un joueur. Cooldown 24h. Les classes Voleur et Illusionniste modifient ces stats.

📜

Quêtes

/quetes — Chaque quête accomplie monte ton niveau de 1. Chaque niveau donne +0.5% de chance d'obtenir un Super Reiatsu.

🎭

Classes

Chaque classe a un passif automatique et un skill actif à déclencher avec /skill. Choisis ta classe avec /classe.

🎰

Bonus & Loterie

/motsecret pour gagner jusqu'à 1000 Reiatsu. /keylottery pour miser 250 Reiatsu et tenter de les doubler ou gagner une clé Steam.

Classes

🥷

Voleur

Passif : Cooldown de vol réduit de 5h, 67% de chance de réussir un vol.

Skill (CD 12h) : Le prochain vol est garanti, 15% de chance de doubler les gains.

🌀

Absorbeur

Passif : +5 Reiatsu par Reiatsu absorbé.

Skill (CD 24h) : Le prochain Reiatsu absorbé est forcément un Super Reiatsu (+100).

🎭

Illusionniste

Passif : 50% de chance de ne pas perdre de Reiatsu quand volé.

Skill (CD 8h) : Envoie un faux Reiatsu — si quelqu'un l'absorbe, tu gagnes 50 Reiatsu.

🎲

Parieur

Passif : À chaque absorption : 50% de rater, 50% de gagner entre 5 et 12 Reiatsu.

Skill (CD 12h) : Machine à sous — mise 30 Reiatsu. 2 symboles identiques = +100, 3 symboles = +160.
