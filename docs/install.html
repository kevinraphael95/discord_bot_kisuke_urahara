Installation

Comment installer et configurer le bot Kisuke Urahara sur ta machine.

1

Prérequis

Assure-toi d'avoir les outils suivants installés sur ta machine avant de commencer.

Python 3.10+ pip Git Compte Discord Developer
2

Cloner le dépôt

Clone le projet depuis GitHub dans le répertoire de ton choix, puis place-toi dedans.

git clone https://github.com/kevinraphael95/discord_bot_kisuke_urahara.git
cd discord_bot_kisuke_urahara
3

Installer les dépendances

Installe toutes les librairies Python requises listées dans requirements.txt.

pip install -r requirements.txt
4

Configurer les variables d'environnement

Crée un fichier .env à la racine du projet et renseigne les variables suivantes.

COMMAND_PREFIX Préfixe des commandes texte (ex : !!)
DISCORD_APP_ID ID de l'application Discord (Developer Portal)
DISCORD_TOKEN Token secret du bot Discord
GPT_OSS_API_KEY Clé API pour le modèle GPT open-source
ADMIN_PASSWORD Mot de passe pour accéder au panel admin
ADMIN_PORT Port du panel admin Flask (ex : 5050)
ADMIN_SECRET Clé secrète pour les sessions Flask
5

Lancer le bot

Lance le bot directement avec Python :

python bot.py

Ou via le script de démarrage complet (recommandé), qui initialise aussi le tunnel Cloudflare pour le panel admin :

bash start.sh
