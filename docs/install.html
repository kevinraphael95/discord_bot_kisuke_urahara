Guide de déploiement

Installation

Tout ce qu'il faut pour faire tourner Kisuke sur ton propre serveur Discord.

01

Prérequis

Avant de commencer, assure-toi d'avoir :

— Python 3.10+ installé sur ta machine.

— Un bot Discord créé sur le Discord Developer Portal, avec les intentions Message Content et Server Members activées.

— Ton token de bot prêt à être utilisé.

02

Cloner le dépôt

Récupère le code source depuis GitHub :

git clone https://github.com/kevinraphael95/discord_bot_kisuke_urahara.git cd discord_bot_kisuke_urahara
03

Installer les dépendances

Installe toutes les bibliothèques Python requises :

pip install -r requirements.txt
Conseil : Il est recommandé d'utiliser un environnement virtuel Python (python -m venv venv) pour isoler les dépendances.
04

Configurer le token

Crée un fichier .env à la racine du projet et ajoutes-y ton token Discord :

# .env DISCORD_TOKEN=ton_token_ici
Important : Ne partage jamais ton token. Ajoute .env à ton .gitignore.
05

Lancer le bot

Démarre Kisuke avec :

python bot.py

Si tout est correct, le bot apparaîtra en ligne sur ton serveur Discord. Tu peux tester avec !help.

06

Synchroniser les commandes slash

Pour activer les commandes slash Discord, lance dans un salon Discord :

!sync

Les commandes slash seront disponibles après quelques secondes (ou minutes pour le global).
