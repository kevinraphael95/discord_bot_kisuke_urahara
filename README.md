# ⚡ Reiatsu Bot Discord

**Reiatsu Bot** est un bot Discord multifonction inspiré de Bleach.  
Il propose des jeux, des commandes utilitaires, un système de Reiatsu et bien plus encore pour animer votre serveur !

## Fonctionnalités principales

- **Admin** : gestion du bot, heartbeat, redéploiement, modification du Reiatsu.  
- **Bleach** : fiches personnages, compatibilités, quizzes et mini-jeux.  
- **Fun & Random** : mini-jeux, pizza aléatoire, couleurs, calculatrice…  
- **Général** : réactions, messages, affichage d’emojis.  
- **Jeux** : pendu, Motus, Mastermind, Anagramme, Compte est Bon, RPG solo/multi.  
- **Reiatsu** : gestion du Reiatsu, classes, quêtes, shop, mini-jeux de collecte et de vol.  
- **Voiture** : mini-jeux automobiles (garage, courses…).
